### 🚀 如何运行应用

#### 1. 启动后端服务器

```bash
cd personal-story-tracker/backend

# 激活虚拟环境 (如果有的话)
# source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate     # Windows

# 安装依赖 (如果需要)
pip install -r requirements.txt

# 运行数据库迁移 (已完成，但可以再次确认)
python manage.py migrate

# 启动开发服务器
python manage.py runserver 0.0.0.0:8000
```

#### 2. 启动前端服务器

```bash
cd personal-story-tracker/frontend

# 安装依赖
npm install

# 启动开发服务器
npm start
```

#### 3. 访问应用

- 前端: http://localhost:3000
- 后端API: http://localhost:8000/api/

### 📱 使用指南

1. **注册账户**: 访问 http://localhost:3000 并创建新账户
2. **登录**: 使用创建的账户登录
3. **查看仪表板**: 登录后会看到统计信息和快速操作
4. **添加条目**: 
   - 点击"添加条目"按钮
   - 选择类型 (物品/人物)
   - 填写基本信息
   - 设置重要度评分
   - 添加故事内容
5. **管理条目**: 
   - 在条目列表中查看所有条目
   - 使用搜索和过滤功能
   - 点击条目查看详情
   - 编辑条目信息和故事

### 📊 数据模型

#### Entry (条目)
- 基本信息: 类型、标题、描述
- 物品字段: 获得日期、价格、类别、状况
- 人物字段: 关系、认识日期
- 重要度评估: 5个维度的1-10分评分
- 故事内容: 富文本内容
- 标签和分类

#### User (用户)
- 用户名、邮箱、显示名称
- JWT认证支持
- 数据隔离 (每个用户只能看到自己的数据)
