# 🌐 个人故事追踪器 - 完整Web体验指南

## 📋 前置要求

要完整体验Web界面，您需要安装以下软件：

### 1. Node.js 和 npm
- **下载地址**: https://nodejs.org/
- **推荐版本**: Node.js 18.x 或更高版本
- **验证安装**: 
  ```bash
  node --version
  npm --version
  ```

### 2. Python 环境 (已有)
- ✅ Python 3.x (已安装)
- ✅ Django 4.2.7 (已安装)

## 🚀 启动步骤

### 步骤 1: 启动后端服务器 (已完成)
```bash
cd personal-story-tracker/backend
python manage.py runserver 0.0.0.0:8000
```
✅ **状态**: 后端已在 http://localhost:8000 运行

### 步骤 2: 安装前端依赖
```bash
cd personal-story-tracker/frontend
npm install
```

### 步骤 3: 启动前端开发服务器
```bash
npm start
```

### 步骤 4: 访问Web应用
- **前端地址**: http://localhost:3000
- **后端API**: http://localhost:8000/api
- **管理后台**: http://localhost:8000/admin

## 🔧 故障排除

### 前端启动问题
```bash
# 清除缓存
npm cache clean --force

# 删除node_modules重新安装
rm -rf node_modules
npm install

# 使用yarn替代npm
npm install -g yarn
yarn install
yarn start
```

### 端口冲突
- 前端默认端口: 3000
- 后端默认端口: 8000
- 如有冲突，可修改package.json中的启动脚本

### API连接问题
- 确认后端服务器运行在 http://localhost:8000
- 检查CORS设置是否正确
- 验证代理配置 (package.json中的proxy字段)
